SVINA软件介绍 

SVINA 分子对接软件

SVINA 是一个基于 AutoDock Vina 的分子对接工具，提供用户友好的图形界面，支持批量分子对接任务。软件集成了 Open Babel、Meeko 和 AutoDock Vina 等工具，简化了配体准备和结果处理流程。
功能特点

    支持批量对接。
    提供基于 Tkinter 的图形化用户界面（GUI）。
    自动化完成以下任务：
        配体文件格式转换（CDX/SDF 转换为 PDBQT）。
        使用 AutoDock Vina 进行分子对接。
        提取对接打分并可视化分子对接结果。
    自定义对接参数支持。
    生成分子网格图像，便于快速查看结果。

安装与使用

安装步骤
方法 1：通过安装包

    下载 SVINA_0.0.1_Setup.exe 安装文件。
    双击运行安装程序，按照提示完成安装。
    安装完成后，桌面会生成 SVINA 快捷方式，双击启动。


使用说明
图形界面操作

    启动软件后，按照以下步骤进行配置：
        工作目录：选择输出结果的保存路径。建议使用cdx文件所在工作目录
        受体文件：选择 .pdbqt 格式的受体文件。建议由CADD人员提供，一个项目一个receptor.pdbqt文件。
        配体文件：选择 .cdx 格式的配体文件（支持批量处理）。可以cdx文件中绘制多个结构。
        对接参数：选择包含对接网格定义的配置文件。建议由CADD人员提供，一个项目一个receptor.pdbqt文件。
    点击 开始运行 按钮，等待对接任务完成。
    运行完成后，您将在输出目录中找到以下文件：
        对接结果（PDBQT 和 SDF 格式）。
        对接打分 CSV 文件。包含Name,smiles,score信息，可有datawarrior软件打开查看结构。
        分子网格图像（PNG 格式）。包含结构、分子名称、打分信息。
